# Mac 控制端设计规格

概念稿：[mac-control-concept.png](mac-control-concept.png)

## 视觉系统

- 背景：主界面真白 `#ffffff`；侧栏石墨 `#20252a`。
- 强调：朱红 `#ed472e` 仅用于当前阶段、主操作和待处理异常。
- 成功：绿色仅表示已完成，不参与装饰。
- 容器：以细分隔线、列表和开放区域为主，避免层层圆角卡片。
- 字体：系统无衬线字体栈，中文优先 PingFang SC；控件 11–13px，项目标题 30px。

## 组件与交互

- 左侧导航：项目、资产、生产、异常、交付。
- 项目头：项目元数据、上传剧集、开始分析。
- 阶段轨：接入、全剧分析、资产确认、自动生产、质量检查、交付。
- 主工作区：剧集表格；异常队列；异常详情和媒体预览。
- 核心本地交互：切换异常项、标记已处理、开始分析状态反馈。

## 设计资产来源

概念稿和播放器画面均由内置图像生成工具生成；界面文本、控件、列表和状态均为 React 原生元素。
